Artist Statement

This project explores the "alternative futures" branching from my current identity as a software engineer. The main path (Amazon > F5) is linear, but the hypertext links break this linearity, allowing a simultaneous glimpse into parallel lives: the hacker, the educator, the architect, and the athlete. The "hacker" aesthetic represents the code that underlies all these potential realities.
